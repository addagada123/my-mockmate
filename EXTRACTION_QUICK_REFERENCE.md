# Quick Reference: Topic Extraction & Language Detection\n\n## Functions Overview\n\n### 1. `_normalize_skill(skill: str) -> str`\nNormalizes skill names to canonical forms.\n\n**Examples:**\n```python\n_normalize_skill(\"REST API\")    ‚Üí \"rest api\"\n_normalize_skill(\"JavaScript\") ‚Üí \"javascript\"\n_normalize_skill(\"ML\")         ‚Üí \"machine learning\"\n_normalize_skill(\"ReactJS\")    ‚Üí \"react\"\n```\n\n**Variations Handled (30+):**\n- Language aliases: JS‚ÜíJavaScript, Py‚ÜíPython, TS‚ÜíTypeScript\n- Framework aliases: React.js‚ÜíReact, Vue.js‚ÜíVue\n- Tool names: Node‚ÜíNodejs, Mongo‚ÜíMongoDB\n- Database: Postgres‚ÜíPostgreSQL, SQL Server‚ÜíMSSQL\n- Multi-word: ML‚Üímachine learning, AI‚Üíartificial intelligence\n\n---\n\n### 2. `_get_skill_category(skill: str) -> str`\nCategorizes a skill into domain area.\n\n**Categories (7 total):**\n- `frontend` - UI/presentational (React, Vue, HTML, CSS)\n- `backend` - Server-side logic (Python, Java, Node, Django, Spring)\n- `database` - Data storage (MongoDB, PostgreSQL, Redis)\n- `devops` - Infrastructure (Docker, Kubernetes, AWS, CI/CD)\n- `mobile` - Mobile apps (React Native, Flutter, iOS, Android)\n- `machine learning` - ML/AI (TensorFlow, PyTorch, NLP)\n- `data` - Data science (Pandas, Spark, Hadoop)\n- `other` - Unmatched skills\n\n**Examples:**\n```python\n_get_skill_category(\"React\")      ‚Üí \"frontend\"\n_get_skill_category(\"Django\")     ‚Üí \"backend\"\n_get_skill_category(\"MongoDB\")    ‚Üí \"database\"\n_get_skill_category(\"TensorFlow\") ‚Üí \"machine learning\"\n```\n\n---\n\n### 3. `_get_top_topics(questions, fallback, limit=5) -> List[str]`\nExtracts and ranks top topics from resume.\n\n**Inputs:**\n- `questions`: List of question dicts with `topic`, `difficulty` fields\n- `fallback`: List of skill strings from resume\n- `limit`: Maximum topics to return (default 5)\n\n**Algorithm:**\n1. Extract topics from questions\n2. Score by: frequency + difficulty weight (easy=1.0x, med=1.5x, hard=2.0x)\n3. Normalize with `_normalize_skill()`\n4. Merge with fallback skills (avoid duplicates)\n5. Rank by category importance (frontend‚Üíbackend‚Üídatabase‚Üídevops‚ÜíML‚Üídata‚Üíother)\n6. Return top N\n\n**Example:**\n```python\nquestions = [\n    {\"topic\": \"REST API\", \"difficulty\": \"hard\"},\n    {\"topic\": \"REST API\", \"difficulty\": \"medium\"},\n    {\"topic\": \"Database Design\", \"difficulty\": \"easy\"},\n]\nskills = [\"ReactJS\", \"RESTful\", \"MongoDB\"]\n\nresult = _get_top_topics(questions, skills, limit=5)\n# Output: [\"rest api\", \"database design\", \"react\", \"mongodb\"]\n# (rest api ranked highest due to 2 questions + hard difficulty)\n```\n\n**Output:**\n- List of normalized, deduplicated, ranked topic strings\n\n---\n\n### 4. `_detect_programming_languages(questions, skills) -> List[str]`\nDetects programming languages from resume.\n\n**Detection Methods (4-tier):**\n1. **Direct pattern matching** - Python, JavaScript, Java, C++, etc.\n2. **Framework inference** - React‚ÜíJavaScript, Django‚ÜíPython, Spring‚ÜíJava\n3. **Ecosystem detection** - npm‚ÜíJavaScript, pip‚ÜíPython, maven‚ÜíJava\n4. **DSA inference** - \"Algorithms\" topic ‚Üí Python + Java\n\n**Supported Languages (15):**\nPython, JavaScript, TypeScript, Java, C++, C, C#, Ruby, PHP, Go, Rust, Kotlin, Swift, Scala, R\n\n**Supported Frameworks (70+):**\n```\nWeb: React, Vue, Angular, Next.js, Express, Django, Flask, FastAPI, Spring\nML: TensorFlow, PyTorch, Scikit-learn, Pandas\nMobile: React Native, Flutter, Swift, Kotlin\nDevOps: Docker, Kubernetes, Jenkins, Terraform\nBuild: npm, pip, maven, gradle, cargo, gem\n```\n\n**Example 1: Framework Detection**\n```python\nquestions = [{\"question\": \"How does React component lifecycle work?\"}]\nskills = [\"Django\", \"FastAPI\"]\n\nlangs = _detect_programming_languages(questions, skills)\n# Output: [\"java\", \"javascript\", \"python\"]\n# Detection:\n# - React ‚Üí JavaScript\n# - Django, FastAPI ‚Üí Python\n```\n\n**Example 2: DSA Inference**\n```python\nquestions = [{\"topic\": \"Data Structures & Algorithms\"}]\nskills = []  # No skills provided\n\nlangs = _detect_programming_languages(questions, skills)\n# Output: [\"java\", \"python\"]\n# Logic: DSA topic detected, no language found ‚Üí default to Python + Java\n```\n\n**Output:**\n- Sorted list of unique language strings\n\n---\n\n## Integration Points\n\n### Location: `/upload-resume` Endpoint\n\n```python\n@app.post(\"/upload-resume\")\nasync def upload_resume(file: UploadFile):\n    # ...
